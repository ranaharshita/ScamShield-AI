# 📖 About The Project

Every day, millions of people receive phishing messages, fake job offers, malicious links, and online scams. Most users realize they are being scammed only after losing money or sensitive information.

**ScamShield AI** is built to solve this problem by providing a simple and accessible platform that can analyze suspicious content and warn users before they become victims.

The platform combines:

* 🤖 Artificial Intelligence
* 🧠 Rule-Based Detection
* 🔒 Privacy-First Design
* ⚡ Real-Time Analysis

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/your-username/scamshield-ai.git
cd scamshield-ai
```

## Install Dependencies

```bash
npm install
```

## Setup Environment Variables

Create `.env.local`

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
GEMINI_API_KEY=
```

## Run Locally

```bash
npm run dev
```
